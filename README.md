# 🎮 GamingZone

GamingZone es una página web enfocada en el mundo gamer, donde los usuarios pueden explorar videojuegos populares, eventos eSports y contenido relacionado con gaming.

## 🚀 Características

- 🎠 Carrusel interactivo de videojuegos
- 🕹️ Sección de top juegos
- 🏆 Eventos y torneos eSports
- 💰 Apartado para venta de videojuegos
- 📱 Diseño moderno y dinámico

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
